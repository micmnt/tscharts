# PieChart

Example of Pie Chart using the Pie component

#### Example

<Canvas of={Pie.Simple} />

## Usage

```ts
import { Pie, Chart } from "tscharts";

const Example = () => {
  return (
    <Chart>
      <Pie name="serie1" />
    </Chart>
  );
};

export default Example;
```

#### Arguments

- **name** - A string that represents the serie associated with pie slices
